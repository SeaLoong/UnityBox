# UnityBox

此仓库用于存放开发中或可直接复用的 Unity 工具脚本，便于在多个项目间共享。

- 想要在你的 Unity 项目中使用某个 Editor 工具：将 `Assets/SeaLoong's UnityBox/` 下的脚本复制到你的项目或把对应的 Package 导入。

## 工具文档

- Blendshape Controller Generator docs: `Docs/BlendshapeControllerGenerator.md`
